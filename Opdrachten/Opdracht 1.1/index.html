Wat is Bootstrap?

Bootstrap is een open-source framework voor het ontwikkelen van responsieve en mobiele-vriendelijke websites.

Met behulp van Bootstrap kunnen we snel een gestructureerde layout maken zonder veel CSS te schrijven.

Voordelen van Bootstrap

  • Responsief ontwerp
  • Gebruiksvriendelijke componenten
  • Snel aan de slag met een gestandaardiseerde opmaak
